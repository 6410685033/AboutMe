RoverC

RoverC Arduino with Raspberry PI and Jetson Nano

RoverC is a Mecanum wheel robot that using M5Stick to control it. In this project, we use Raspberry PI to control RoverC by sending data using MQTT Protocol. By using Leafhat, we can use joystick PI to send data to M5Stick. We add object detection for detect RoverC and destination using Jetson Nano and that’s it, RoverC can move now.
